# Help Desk Lab

Laboratório pessoal de suporte técnico e troubleshooting.

Este repositório reúne procedimentos, diagnósticos e soluções para problemas comuns encontrados em suporte técnico e Help Desk, utilizados para estudo, prática e desenvolvimento profissional na área de TI.

## Conteúdo

### Windows
- Lentidão no computador
- Problemas de internet
- Erros de inicialização
- Impressoras
- Tela azul (BSOD)

### Redes
- Diagnóstico de conectividade
- Comandos de rede
- Testes de conexão

### Hardware
- Checklist de manutenção
- Diagnóstico de componentes

### Scripts
- Automação simples para troubleshooting

## Objetivo
Desenvolver conhecimentos práticos em suporte técnico, troubleshooting e infraestrutura de TI, com foco em atuação como Help Desk e Analista de TI Júnior.
